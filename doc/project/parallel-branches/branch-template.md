# Parallel Branch Template

Use this template when assigning a new Claude/Gemini/Codex branch.

## Branch

- `branch/name-here`

## Purpose

One short paragraph describing the branch goal.

## Read First

- `AGENTS.md`
- `doc/project/playerbot-foundation-handoff.md`
- `doc/project/playerbot-collaborator-brief.md`
- `doc/project/parallel-branch-workmap.md`
- any task-specific docs

## Allowed Files

- list exact owned files or directories

## Forbidden Files

- list exact hotspots that must not be touched

## Task

- flat list of the exact deliverables

## Validation

- exact commands to run
- exact success criteria

## Reporting Back

Contributor must report:

- what changed
- what was validated
- commit hash
- whether the branch is proposed as merge-ready

## Merge Conditions

The branch is not accepted until:

- reviewed against current `master`
- scope is clean
- validation passes
- docs and labels match reality
